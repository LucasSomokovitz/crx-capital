# Pasta de Imagens

Esta pasta contém as imagens estáticas utilizadas na landing page da CRX Capital.

## Imagens Necessárias

### 1. hero-team.jpg
- **Uso**: Imagem de fundo da seção Hero
- **Dimensões recomendadas**: 1920x1080px (Full HD)
- **Formato**: JPG ou WebP
- **Descrição**: Foto profissional da equipe em reunião ou ambiente corporativo
- **Otimização**: Comprimir para web (máximo 500KB)

### 2. about-office.jpg
- **Uso**: Imagem da seção "Sobre a CRX Capital"
- **Dimensões recomendadas**: 800x800px (quadrado) ou 1200x800px
- **Formato**: JPG ou WebP
- **Descrição**: Foto do escritório, fachada ou equipe trabalhando
- **Otimização**: Comprimir para web (máximo 300KB)

### 3. logo.png
- **Uso**: Logo da empresa no header e footer
- **Dimensões recomendadas**: 200x60px (formato horizontal)
- **Formato**: PNG com fundo transparente
- **Descrição**: Logo oficial da CRX Capital
- **Otimização**: PNG otimizado ou SVG

## Dicas de Otimização

- Use ferramentas como [TinyPNG](https://tinypng.com/) ou [Squoosh](https://squoosh.app/) para comprimir imagens
- Prefira formato WebP quando possível (melhor compressão)
- Mantenha imagens abaixo de 500KB para melhor performance
- Use imagens responsivas com diferentes tamanhos para mobile/desktop se necessário

## Estrutura Sugerida

```
public/images/
├── hero-team.jpg
├── hero-team-mobile.jpg (opcional - versão mobile)
├── about-office.jpg
├── logo.png
└── logo-white.png (opcional - versão branca para footer escuro)
```
